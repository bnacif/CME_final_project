# CME_final_project
This project solves the model in "Capital Flows and Informality", an extension of Schmitt-Grohé and Uribe (2016) with formal and informal labor.

To run the code you need to follow the steps:

1. Run `init_packages.jl` only the first time.

2. Run `main.jl`.

All comments are in the code files. Results of the simulation are discussed in CME_final_project.pdf.
